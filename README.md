# Fake Python Jobs Scraper

A simple Python web scraper that collects job listings from the Fake Python Jobs website.

## Features

- Scrapes job title
- Scrapes company name
- Scrapes location
- Scrapes job detail URL
- Saves all data into a CSV file

## Technologies

- Python
- Requests
- BeautifulSoup4
- CSV

## Installation

Clone this repository.

```bash
git clone https://github.com/yourusername/fake-python-jobs-scraper.git
```

Install dependencies.

```bash
pip install -r requirements.txt
```

Run the scraper.

```bash
python scraper.py
```

The scraped data will be saved into:

```
jobs.csv
```
